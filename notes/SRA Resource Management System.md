Web application to manage clients, projects, resources and resource allocations.

### Key features
___
1. CRUD operations on [[Client]]s, [[Project]]s, [[Resource]]s (people) and resource allocations
2. Management dashboard
3. Visualize resource allocation using Gantt charts
4. Reporting

### Technology
___
1. Front end: VueJS
2. Business layer: C#
3. Data layer: Postgres (localhost:5432, user: postgres, password: Srasra44#)

### Authentication
___
Microsoft active directory

### Authorization
___
Role base authorization withing the application. Three main roles
1. Administrator: CRUD operations
2. General: View data
3. Report: View reports
